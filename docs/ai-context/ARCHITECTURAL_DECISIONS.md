# Architectural Decisions

- OpenClaw remains an orchestrator/interface, not the core domain brain.
- Obsidian remains the durable knowledge/documentation layer.
- SQLite remains the operational structured store for now.
- Conversation is an interface, not the center of the platform.
- The system is evolving into a reusable mother project.
- Monitoring, alerts, workflows, and reusable APIs are first-class future directions.
